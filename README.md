Deskripsi Aplikasi:
ZenTimer adalah aplikasi pengatur waktu (timer) berbasis mobile yang dikhususkan untuk kegiatan relaksasi dan meditasi. Berbeda dengan aplikasi timer biasa, ZenTimer mengintegrasikan pengaturan waktu dengan pemutar suara latar (ambience noise) bertema alam yang menenangkan. Desain antarmuka aplikasi ini menggunakan default tema gelap (dark theme) untuk memberikan kenyamanan visual dan mendukung suasana rileks, terutama saat digunakan di malam hari atau ruangan redup.
